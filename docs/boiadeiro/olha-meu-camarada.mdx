# Olha meu camarada

### Letra

(2x)
Olha meu camarada
Camarada meu

A Umbanda é uma escola

Aqui dentro eu encontrei
Minha família de verdade

(2x)
Olha meu camarada
Camarada meu

Boiadeiro sambador

(2x)

Camarada meu

Estremece todo chão

(2x)
Olha meu camarada
Camarada meu
