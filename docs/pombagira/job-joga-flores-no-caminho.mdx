# Job joga flores no caminho

### Letra

```text
Job joga flores no caminho
Não me deixe andar sozinho
Nesse mundo de meu Deus

Você que é uma rainha
Mas deixou tudo que tinha
Pra viver com os plebeus

(2x)
Você que é uma rainha

Você é Maria de Job
Para nós tu és Mulambo
```
