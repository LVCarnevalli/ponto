# Cemitério é praça linda

### Letra

```text
Cemitério é praça linda, mas ninguém quer passear
Lá tem sete catacumba, a Padilha mora lá
Mora lá, mora lá, mora lá, mora lá
Cemitério é praça linda, mas ninguém quer passear
Lá tem sete catacumba, a Cigana mora lá
Mora lá, mora lá, mora lá, mora lá
```
