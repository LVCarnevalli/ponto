# Cruzeiro das almas

### Letra

```text
(2x)
Nasceu no cruzeiro das almas
Uma roseira que já deu flor
Entre elas um rosa
Que em uma linda mulher
Se transformou

(2x)
Praticando sua caridade com muito amor
Exalando harmonia com o perfurme da flor

(2x)
Ela é moça bonita
É faceira e formosa
Pombagira do Cruzeiro
É a mais bela das rosas
```
