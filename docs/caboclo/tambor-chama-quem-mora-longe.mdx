# Tambor, chama quem mora longe

### Letra

```text
Tambor tambor
Chama quem mora longe
Tambor tambor
Chama quem mora longe

A gira é de Caboclo
Foi Oxalá quem mandou
Segura o couro tambor
Que ele vai chegar
```
