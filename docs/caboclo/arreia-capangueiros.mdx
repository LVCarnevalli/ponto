# Arreia capangueiros

### Letra

```text
(2x)
Arreia capangueiros, os capangueiros da Jurema
Arreia capangueiros, os capangueiros Juremá

(2x)
Se não fosse os capangueiros
O que seria da Jurema
Se não fosse os capangueiros
O que seria o Juremá

(2x)
Olha o mato quebrando
Caboclo arriando
```
