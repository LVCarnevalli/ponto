# Ela é oyá

### Letra

```text
Olha que o céu clareou
Quando o dia raiou
Fez o filho pensar

A mãe do tempo mandou
A nova era chegou
Agora vamos plantar

Do Humaitá Ogum bradou
Senhor Oxossi atinou
Iansã vai chegar

O ogã já firmou
Atabaque afinou
Agora vamos cantar

Ah! Eparrei!
Ela é Oyá! Ela é Oyá!
Ah! Eparrei!
É Iansã! É Iansã!
Ah! Eparrei!

Quando Iansã vai pra batalha
Todos cavaleiros param
Só pra ver ela passar.

Ah! Eparrei!
Ela é Oyá! Ela é Oyá!
Ah! Eparrei!
É Iansã! É Iansã!
Ah! Eparrei!

Quando Iansã vai pra batalha
Todos cavaleiros param
Só pra ver ela passar
```
