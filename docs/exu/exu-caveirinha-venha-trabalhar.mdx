# Exú Caveirinha venha trabalhar

### Letra

```text
Exú Caveirinha
Venha trabalhar
Levanta dessa Tumba
Faz pedra rolar

Na mão esquerda a foice
Na cinta o punhal
Não sai da linha mano
Pra não se dar mal
```
