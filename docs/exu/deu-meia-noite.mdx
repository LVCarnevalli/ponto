# Deu meia noite

### Letra

```text
Deu meia noite
A Lua se escondeu
Lá na encruzilhada, dando a sua gargalhada
Tranca Rua apareceu

(2x)
É laroyê, é laroyê, é laroyê
É mojubá, é mojubá, é mojubá
Ele é odara, dando a sua gargalhada
Quem tem fé em Tranca Rua, é só pedir, que ele dá
```
