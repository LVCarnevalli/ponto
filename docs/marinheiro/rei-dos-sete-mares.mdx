# Rei dos sete mares

### Letra

```text
Seu Marinheiro é o rei dos sete mares

Matou tubarão, enfrentou piratas
Ganhou de Iemanjá uma coroa de prata

Ele é o rei dos sete mares (2x)

(2x)
Que chegou no terreiro
Pra levar nossos males
```
