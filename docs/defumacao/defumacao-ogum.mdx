# Defumação Ogum

### Letra

```text
Corre gira pai Ogum
Filhos quer se defumar
Umbanda tem fundamento
É preciso preparar
Com incenso e benjoim
Alecrim e Alfazema
Defumar filhos de fé
Com as ervas da Jurema
```
